# Pesquisa-Operacional---Programa-o-Linear
Trabalho de Pesquisa Operacional 
Objetivo do trabalho
Implementar um algoritmo em uma linguagem de programação que resolva um
problema obrigatoriamente relacionado com o conteúdo da disciplina de
Pesquisa Operacional e que se aproxime de uma situação real, aplicando um
dos seguintes temas: Programação linear, Problema de transporte ou Problema
de designação.

O tema escolhi para resolução do problema foi Programação Linear 

Optei por Programação Linear devido à sua relevância e aplicabilidade em diversos cenários

A Programação Linear (PL) é um método matemático utilizado para otimizar o desempenho de um sistema, dado um conjunto de restrições. Ela lida com problemas nos quais se busca maximizar ou minimizar uma função linear, sujeita a um conjunto de restrições lineares. A função a ser otimizada é chamada de função objetivo, e as restrições são as condições que o sistema deve satisfazer.
